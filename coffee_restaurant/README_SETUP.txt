COFFEE RESTAURANT WEBSITE (PHP + MySQL) - FULL PROJECT
=====================================================

RUNNING STEPS (XAMPP):
1) Install XAMPP
2) Start Apache + MySQL from XAMPP Control Panel
3) Copy the folder "coffee_restaurant" into:
   C:\xampp\htdocs\

4) Open phpMyAdmin:
   http://localhost/phpmyadmin

5) Create database and tables:
   - Create a DB named: coffee_restaurant
   - Import the file: database.sql

6) Open the website:
   http://localhost/coffee_restaurant/

ADMIN PANEL:
- URL: http://localhost/coffee_restaurant/admin/login.php
